# Run React project 运行React项目
The React project is built with React + Vite + Tailwind CSS and cannot be opened by double-clicking an HTML file. To run it locally, make sure you have Node.js installed, then install the dependencies and start the development server:

```bash
npm install
npm run dev
```

After running the above commands, open the URL shown in your terminal (usually http://localhost:5173/) in your browser.
Some game may requires **camera access**, so please allow your browser to use the webcam.
